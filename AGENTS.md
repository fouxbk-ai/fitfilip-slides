# Правила работы с репозиторием

- Источник истины — GitHub `main`; публикация выполняется GitHub Pages.
- Каждая доработка идёт через Issue, ветку `codex/<slug>`, PR и зелёный CI.
- JSON/HTML должны оставаться статическими, без секретов и private API keys.
- Перед merge проверять ссылки из `index.html` и парсинг всех JSON-файлов.
